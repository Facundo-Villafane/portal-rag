2. División de un PNR

También se le dice PNR Split, sirve para separar pasajeros que están dentro de un mismo PNR porque necesitan vuelos diferentes.

Comando SP

SP + numero del pasajero que queremos separar

Para hacer un Split debemos:

  1. Abrir el RL que quiero dividir con RT
  2. Usar el comando SP para dividir
  3. Firmar con RF
  4. Cerrar el PNR para que me cree un nuevo, el cierre de elementos Split es con EF
  5. Firmamos nuevamente con RF y esta vez si usamos ET para que nos genere un nuevo RL (del Split)
  6. Traemos el nuevo PNR con RT
  7. Y hacemos los cambios que sean necesarios.
